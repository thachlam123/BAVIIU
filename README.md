# Anh Em Ba Vì — Immersive Motion Website V4

Bản V4 đã sửa theo yêu cầu:

- Đổi phần chữ sang tiếng Việt tự nhiên hơn.
- Hiện con trỏ thật của máy để dễ bấm hơn, vẫn giữ vòng cursor neon chạy theo chuột.
- Nút vào web rõ hơn: **BẤM VÀO WEB • BẬT NHẠC**.
- Nhạc nền chuyển sang MP3 và chạy bằng thẻ audio ẩn trong website, không hiện thành link tải.
- Nhạc sẽ phát sau cú bấm đầu tiên vì trình duyệt chặn autoplay có tiếng nếu người dùng chưa tương tác.
- `run.bat` tự mở trình duyệt tới `http://localhost:5173`.

## Cách chạy

1. Giải nén thư mục.
2. Bấm đúp `run.bat`.
3. Trình duyệt mở `http://localhost:5173`.
4. Bấm nút **BẤM VÀO WEB • BẬT NHẠC**.

## Ghi chú

Không nên mở trực tiếp file `index.html` bằng `file://` vì trình duyệt có thể chặn hoặc xử lý sai nhạc/video. Hãy chạy bằng `run.bat` hoặc upload lên hosting/GitHub Pages.
